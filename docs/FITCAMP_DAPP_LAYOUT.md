## 设计要点（DApp 常见风格）

| 区域 | 说明 |
|------|------|
| **顶栏** | 左侧品牌，中间链/网络标识（可点击切链），右侧钱包连接/已连接地址 + 断开，固定或吸顶 |
| **内容区** | 单列、最大宽度约 420–480px 居中，卡片间距统一，圆角 + 浅边框或深色背景卡片 |
| **卡片** | 每块功能一个卡片（当前期、我的参与、群主操作、Fit NFT、状态），主操作按钮醒目 |
| **层级** | 主操作（打卡、提现、开始/结束打卡）用主色按钮；次要信息（状态、参与名单）弱化或可折叠 |
| **未连接** | 顶栏显示「连接钱包」主按钮；连接后显示短地址 + 断开，地址可做成按钮形态 |
